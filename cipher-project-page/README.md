# CIPHER Project Page

Project website for **CIPHER** (CVPR 2026): *Fighting Hallucinations with Counterfactuals: Diffusion-Guided Perturbations for LVLM Hallucination Suppression*.

This repository is the source for the project webpage, intended to be hosted with **GitHub Pages**.

## Deploy with GitHub Pages

1. **Push this folder’s contents** to your website repo:  
   [https://github.com/hamidreza-dastmalchi/cipher-cvpr2026](https://github.com/hamidreza-dastmalchi/cipher-cvpr2026)

2. In the repo: **Settings → Pages**.

3. Under **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or your default branch)
   - **Folder:** `/ (root)` if `index.html` is in the repo root, or **/docs** if you put the site in a `docs` folder.

4. Save. The site will be available at:  
   **https://hamidreza-dastmalchi.github.io/cipher-cvpr2026/**

## Contents

- `index.html` — Main page
- `style.css` — Styles
- `assets/` — Add `teaser.png`, `pipeline.png`, `results.png` (see `assets/README.md`)

Update the **Paper**, **arXiv**, and **Code** links in `index.html` when you have the URLs.
